# Plan: Analyze Requirements

## Trigger

Auto-detect when user asks to:

- Understand a feature or bug
- Explore a codebase
- Plan implementation
- Break down a task

## Input

- User request
- Current codebase context
- UBIQUITOUS_LANGUAGE.md (if exists)

## Instructions

1. Read the user request carefully
2. Identify the core problem or feature
3. Determine scope and constraints
4. Check for existing similar features
5. Identify dependencies

## Output

- Problem statement
- Scope definition
- Dependencies list
- Risk assessment
